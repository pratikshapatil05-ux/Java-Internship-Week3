# Java-Internship-Week3
This week focuses on understanding Object-Oriented Programming (OOP) in Java.
You will learn how to create classes, objects, constructors, encapsulated data, and simple class relationships.

1. Theory Concepts Covered
* Introduction to OOP

What is Object-Oriented Programming

Real-world modeling using objects

OOP pillars (Encapsulation, Abstraction, Inheritance, Polymorphism)

* Classes and Objects

Class = blueprint

Object = real entity created from class

How to define attributes (variables) and behaviors (methods)

* Constructors & Method Overloading

Default constructor

Parameterized constructor

Constructor initialization

Method overloading (same method name, different parameters)

* Access Modifiers
Modifier	Access
public	Accessible everywhere
private	Within same class only
protected	Same package + subclasses
default	Same package
* Encapsulation & Data Hiding

Variables declared as private

Accessed using getters and setters

Protects data from unauthorized access

* Basic Class Relationships
Has-a relationship
Uses-a relationship
Simple object interactions

 5. Learning Outcomes

By completing Week 3, you will be able to:

* Create classes and objects
* Use constructors (default + parameterized)
* Apply method overloading
* Use access modifiers (public/private/protected)
* Implement encapsulation with getters and setters
* Build interconnected classes for real-world problems
* Understand how OOP improves code structure
